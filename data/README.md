# Data

This project uses the following datasets:

- `2017_Yellow_Taxi_Trip_Data.csv`
- `nyc_preds_means.csv`

The raw data files are not included in this repository by default.

Place both files inside this `data/` folder before running the notebooks.

Expected structure:

data/
- 2017_Yellow_Taxi_Trip_Data.csv
- nyc_preds_means.csv
